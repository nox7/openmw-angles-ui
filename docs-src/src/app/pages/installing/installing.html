Installing AnglesUI - for Mod Developers

AnglesUI is a UI framework for you to create UIs in your OpenMW mod or addon. By itself, it does not modify or inject itself into the game. Simply put, installing this and then going to play OpenMW will not have any effect. You have to actually use it in your mods.

Download from the GitHub releases.

Once you have the latest release, you should use the recommended OpenMW mod installation approach. If you don't already, add a folder named mods into your OpenMW game data directory (On Windows, this is in Documents/My Games/OpenMW). Extract the download into that folder. Then, run the OpenMW launcher and click the Data Directories tab. Add the new folder into the data directories. Now you can use it in your mods.
