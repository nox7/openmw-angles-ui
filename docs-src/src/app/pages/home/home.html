About AnglesUI

AnglesUI is a full-featured UI framework, templating syntax, and rendering engine built on top of OpenMW's Lua UI API. AnglesUI requires no dependencies — it is fully implemented in plain Lua 5.1 syntax. The rendering engine is a port of parts of Chromium's Blink rendering engine to Lua.

This framework comes built in with an HTML (with Angular-style directives support) parser, CSS parser, and Blink-ported layout rendering engine. This means you can use HTML and CSS to create your OpenMW UIs and statefully manage them with signals, so you no longer need to build or maintain UIs in Lua for your mod.

Because AnglesUI supports CSS media queries and container queries — along with a rendering engine that is per-frame accurate — you can create responsive UIs that work on all screen sizes without writing any Lua code.

This project was born due to a need in my own mod for a UI that was responsive, easy to make and maintain without Lua code, and be designed and built in under 30 minutes.
