The AnglesUI Framework

AnglesUI is a full-featured UI framework, templating syntax, and rendering engine built on top of OpenMW's Lua UI API. AnglesUI requires no dependencies. It is fully implemented in plain Lua 5.1 syntax.

This framework comes built in with an HTML (with Angular-style directives support) parser, CSS parser, and Blink-ported layout rendering engine. This means you can use HTML and CSS to create your OpenMW UIs and statefully manage them with signals so you no longer need to build or maintain UIs in Lua for your mod.

Additionally, because AnglesUI supports CSS media queries and container queries (along with our rendering engine being per-frame accurate) you can create responsive UIs that work on all screen sizes without having to write any Lua code.
