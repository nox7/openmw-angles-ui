Window Element

A pre-styled (to Morrowind style) element that automatically expands to fit its parent element.

Selector

<mw-window>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.

CSS Properties

Property
Data Type
Description
 width pixels or percentage The width of the element. height pixels or percentage The height of the element. visibility "hidden" or "visible" The visibility of the element. padding pixels Padding inside the element. flex-grow number Specifies how much the element will grow relative to the rest of the flexible items inside the same container. See MDN docs. background "none" Only accepts "none". Removes the background from the window element and leaves only the borders.
