Grid Element

An implementation of the Chromium Blink rendering engine's grid. When you know you will need multiple rows and columns at a given screen size, choose a grid element over a flex element.

Selector

<mw-grid>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.

Property
Data Type
Description

A number between 0 and 1. aspect-ratio number or division Forces the element to maintain the aspect ratio. This is ignored if you define both a width and height. For this to work, define only one. Note: may not function as expected, or at all, on direct grid or flex children. grid-template-columns string Defines the number of columns and their size in the grid.
