Scroll Canvas Element

A scroll canvas is a UI implementation that allows for a canvas and a viewport to be set up such all child elements can be scrolled horizontally or vertically when they overflow from the scroll canvases width or height. By default, it is Morrowind-UI styled.

It can contain any type of child element - including windows and images, and even other scroll canvases.

Selector

<mw-scroll-canvas>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.
