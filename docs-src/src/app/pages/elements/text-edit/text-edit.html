Text Edit Element

A user text input element. Allows the user to type text in. Adds additional non-OpenMW-standard support for a Placeholder attribute. By default, the text in the input is styled for a Morrowind-UI.

This element does not support padding. If you need padding, wrap it in an <md-widget> element and add padding to that widget.

Selector

<mw-text-edit>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Placeholder string The placeholder text displayed when the input is empty. MultiLine boolean Whether the text element supports multiple lines. Does not work when AutoSize is true. WordWrap boolean Whether the text element supports word wrapping. Does not work when AutoSize is true. AutoSize boolean Defaults to false. Automatically sizes the element based on the text size in it. Enabling this will not allow width or height to be set. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.
