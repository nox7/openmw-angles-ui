Horizontal Rule (HR) Element

A horizontal rule (HR) element is a UI element that visually separates content. It is commonly used to create a thematic break between sections of content. By default, it is Morrowind-UI styled.

Also, by default, it will automatically expand its width to its parent container width and be 2 pixels tall.

This element does not support padding. If you need padding, wrap it in an <md-widget> element and add padding to that widget.

Selector

<mw-hr>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.
