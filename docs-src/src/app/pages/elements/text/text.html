Text Element

A basic element that renders text. The text content must be a child text node. As in, between the HTML tags of the element. There is no HTML attribute for the text content.

This element does not support padding. If you need padding, wrap it in an <md-widget> element and add padding to that widget.

Text is auto-sized based on the text inside the element by default. This means you cannot set a height or width on this element. Turn AutoSize off by setting its attribute to false to use custom sizing.

Selector

<mw-text>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. AutoSize boolean Defaults to true. Automatically sizes the element based on the text size in it, but can be turned off to use custom element width and height. MultiLine boolean Whether the text element supports multiple lines. Note: only functions correctly when AutoSize is false. WordWrap boolean Whether the text element supports word wrapping. Note: only functions correctly when AutoSize is false. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.
