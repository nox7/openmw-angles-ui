Flex Element

An implementation of the Chromium Blink rendering engine's flexbox. It's important to note that this does not use OpenMW's "flex" element. This is a custom implementation.

Selector

<mw-flex>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element.

CSS Properties

Property
Data Type
Description
 width pixels or percentage The width of the element. height pixels or percentage The height of the element. visibility "hidden" or "visible" The visibility of the element. padding pixels Padding inside the element. opacity number How opaque the element is. A number between 0 and 1. aspect-ratio number or division Forces the element to maintain the aspect ratio. This is ignored if you define both a width and height. For this to work, define only one. Note: may not function as expected, or at all, on direct grid or flex children. flex-direction "column" or "row" Sets the major axis that this element's children will be aligned on. Only "column" and "row" are supported and not the reverse variants. gap pixels Specifies the gap between the children of this element. flex-grow number Specifies how much the element will grow relative to the rest of the flexible items inside the same container. See MDN docs. Only works if the parent element is a flex element. grid-column mixed string If the parent is a grid element, this specifies the column this element takes and/or how many it spans. See MDN docs. We accept only the formats of "1" or "1 / span 2" as examples. grid-row mixed string If the parent is a grid element, this specifies the row this element takes and/or how many it spans. See MDN docs. We accept only the formats of "1" or "1 / span 2" as examples.
