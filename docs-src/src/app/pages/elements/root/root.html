Root Element

This is the required root element in a parent-level component. It must always be the first element in your hierarchy for the main file that is passed to the renderer.

Selector

<mw-root>

Attributes

Attribute
Data Type
Description
 id string The HTML element id. Used in CSS selectors. class string The HTML element class string. Used in CSS selectors. Layer string The UI layer this root will be rendered onto. 1-to-1 to the OpenMW UI layers. Resizable boolean Implements the behavior that when the user drags the edge or corner of the root element, it resizes with the drag operation. Dragger boolean Implements the behavior that when the user drags this element, the root element will be moved with the cursor. Works on any UI element, but will only ever drag the root element. EdgeMargin pixels The margin at the edges of the root element that will be considered for resizing operations when the user clicks within that threshold and drags.

CSS Properties

Property
Data Type
Description
 width pixels or percentage The width of the element. height pixels or percentage The height of the element. visibility "hidden" or "visible" The visibility of the element.
