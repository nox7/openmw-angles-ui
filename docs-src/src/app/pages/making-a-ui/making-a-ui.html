Making a Basic UI With AnglesUI

Now that you have installed AnglesUI, you can start making your UIs with enhanced-HTML and CSS. We write enhanced here because the HTML syntax is expaned to be a template language that can support if statements, for loops, and variable output directives.

File and Folder Structure

We recommend creating a folder dedicated to your UIs in your mod directory. This is where your HTML and CSS files will live for your Lua code to load and render them when you need them.
