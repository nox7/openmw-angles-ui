All Valid HTML Elements

This is a list of all of the HTML elements that AnglesUI supports and what they do and any valid attributes on them.

  • mw-root: The root element of an AnglesUI rendered parent.
  • mw-window: A pre-styled Morrowind-UI window. Automatically has height 100% and width 100% of its parent.
  • mw-text: A text element pre-styled to use Morrowind-style text. Has OpenMW UI autoSize set to true by default.
  • mw-widget: A generic widget element with no styling. Think of it as a base container for styling or elements. In the HTML world, this would be your <div>
  • mw-flex: A custom flexbox implementation. Used for putting your elements in rows or columns, but not usually both.
  • mw-grid: A custom grid implementation. Used for creating more complex grid-like layouts with multiple cells in rows and column format.
  • mw-image: Renders an image resource.
  • mw-text-edit: A text input field for the user to type into. Listen for the textInput or textChanged events to react to user input.
  • mw-hr: A horizontal line styled in Morrowind-UI theme.
  • mw-scroll-canvas: An element that provides a scrollable canvas area for arbitrary content. Works for both vertical and horizontal scrolling. Styled in Morrowind-UI theme.

You may be wondering why you don't see something like a button, for example. The reason is because AnglesUI is a framework for you to make your own components - with some pre-configured for a "batteries partially included" effect.
